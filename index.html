What do I do?

I code.

I'm a full-stack application developer with a particular interest in scalable systems. I've written everything from an operating system kernel in C to a single-page web app in React. I love mastering new technologies. My primary expertise is in C# and systems architecture.

I organize.

My happy place includes large quantities of well-organized code. My heart leaps for AGILE development cycles, UML code diagrams, and short CI/CD release trains. My vacations feature color-coded spreadsheet timetables. My family runs on a bullet journal. Organization is something I do at my core.

I create.

I draw; I sculpt; I build server racks and deploy Windows domains on virtualized servers. I go on the occasional crafting hyperfixation. If it involves making something or learning how to make something, I've probably tried it.

Pipesim GIS

Schlumberger's PIPESIM team needed a toolkit which could render a large pipeline network over a map to replace existing logic. I worked extensively with Schlumberger's architects to design and implement a toolkit which would fill their feature requirements while remaining maintainable, extensible, and reliable. I also setup TFS builds which published the toolkit to the company's internal NuGet repository, worked with management on team coordination and deadlines, and setup an automation framework and automated tests for catching regressions.

Sociyall

Startup Neuric wanted to showcase their comprehensive semantic network and it's application to data analytics at Austin's SXSW conference with a content aggregation app. I lead the app development team and contributed to development by completing the middle-layer logic, code architecture and database architecture. By working with a third-party REST API, I pulled in social media posts written as JSON data to an AWS repository. I ran the information through Neuric's proprietary, C++ parser and transformed the results for the PostGreSQL back-end.

  • REST

  • SQL
  • C#
  • Entity Framework
  • Performance Profiling
  • UML

As a personal project, my husband and I put together a desktop application in order to play a popular collectible-card game.
